Utility Functions

Unit step function

A unit step function is defined by

\[u(t) = \begin{cases} 1, & t \geq 0 \\ 0, & t < 0 \end{cases}.\]
using ISA
t = 1.25
x = u(t-0.25)-u(t-2.5)
1.0

Gaussian function

A Gaussian function is defined by

\[f(x) = \mathrm{e}^{-0.5\left(\dfrac{x-μ}{σ}\right)^2}\]

where $x$ is an input data, μ is mean and σ is the standard deviation.

using ISA
x = 2.0
μ = 1.0
σ = 0.1
f = 𝒩ᵤ(x::Float64; μ, σ)
1.9287498479639178e-22

Normalized Gaussian function

A normalized Gaussian function is defined by

\[f(x) = \dfrac{1}{√(2π)σ}\mathrm{e}^{-0.5\left(\dfrac{x-μ}{σ}\right)^2}\]

where $x$ is an input data, μ is mean and σ is the standard deviation.

using ISA
x = 2.0
μ = 1.0
σ = 0.1
f = 𝒩(x::Float64; μ, σ)
7.694598626706419e-22

Dirac delta function

A Dirac delta function is defined using Gaussian approximation with zero mean and variance $σ^2$ tending to zero (in limit).

\[δ(t) = \mathrm{e}^{-0.5\left(\dfrac{t^2}{σ^2}\right)}.\]
using ISA
t = 1.0e-10
σ = 1.0e-11
x = δn(t, σ)
1.9287498479638767e-22

Another representation of Dirac delta function is defined using normalized Gaussian approximation with zero mean and variance $σ^2$ tending to zero (in limit). This definition truly represents the Dirac delta function in approximation.

\[δ(t) = \dfrac{1}{√(2π)σ}\mathrm{e}^{-0.5\left(\dfrac{t^2}{σ^2}\right)}.\]
using ISA
t = 1.0e-10
σ = 1.0e-11
x = δ(t, σ)
7.694598626706256e-12
