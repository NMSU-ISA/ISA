AM–FM Models

A component set maps to an AM–FM model AMFMmodel.

\[z\left( t ; \mathscr{S} \vphantom{0^0}\right) \triangleq \sum\limits_{k=0}^{K-1}\psi_k\left( t ; \mathscr{C}_k \vphantom{0^0}\right),~\mathscr{S}\triangleq\left\{\mathscr{C}_0,\mathscr{C}_1,\cdots,\mathscr{C}_{K-1}\vphantom{0^0}\right\}\]

This mapping is provided in the ISA module.

Defining an AM–FM Model

We define an AM–FM model primarily by passing an object, 𝑆 to the function AMFMmodel(). First, define a component set, 𝑆 by passing a vector of type AMFMtriplet to the function compSet.

using ISA
C₀ = AMFMtriplet(t->cos.(t),ω->100,0.1)
C₁ = AMFMtriplet(t->2*t,ω->10,1.0)

𝑆 = compSet([C₀,C₁])
z = AMFMmodel(𝑆)
AMFMmodel(compSet(AMFMtriplet[AMFMtriplet(Main.##ex-#261.var"#1#3"(), Main.##ex-#261.var"#2#4"(), 0.1), AMFMtriplet(Main.##ex-#261.var"#5#7"(), Main.##ex-#261.var"#6#8"(), 1.0)]))

We also allow an AM–FM model AMFMmodel to be defined by passing an vector of AMFMcomp to the function AMFMmodel().

using ISA

a₀(t) = exp(-t^2)
ω₀(t) = 2.0
φ₀ = 0.0
ψ₀ = AMFMcomp(a₀,ω₀,φ₀)

a₁(t) = 1.0
ω₁(t) = 10*t
φ₁ = 0.1
ψ₁ = AMFMcomp(a₁,ω₁,φ₁)

a₂(t) = 0.8*cos(2t)
ω₂(t) = 10 + 7.5*sin(t)
φ₂ = π
ψ₂ = AMFMcomp(a₂,ω₂,φ₂)

z = AMFMmodel([ψ₀,ψ₁,ψ₂])
AMFMmodel(compSet(AMFMtriplet[AMFMtriplet(Main.##ex-#262.a₀, Main.##ex-#262.ω₀, 0.0), AMFMtriplet(Main.##ex-#262.a₁, Main.##ex-#262.ω₁, 0.1), AMFMtriplet(Main.##ex-#262.a₂, Main.##ex-#262.ω₂, π)]))

There is another method to define AM–FM model AMFMmodel by passing an vector of AMFMtriplet to the function AMFMmodel().

using ISA

C₀ = AMFMtriplet(t->cos.(t),ω->100,0.1)
C₁ = AMFMtriplet(t->2*t,ω->10,1.0)

z = AMFMmodel([C₀,C₁])
AMFMmodel(compSet(AMFMtriplet[AMFMtriplet(Main.##ex-#263.var"#1#3"(), Main.##ex-#263.var"#2#4"(), 0.1), AMFMtriplet(Main.##ex-#263.var"#5#7"(), Main.##ex-#263.var"#6#8"(), 1.0)]))

Evaluating an AM–FM Model

Once an AM–FM model AMFMmodel is defined it can be evaluated at a time instant Float64

using ISA

C₀ = AMFMtriplet(t->cos.(t),ω->100,0.1)
C₁ = AMFMtriplet(t->2*t,ω->10,1.0)

𝑆 = compSet([C₀,C₁])
z = AMFMmodel(𝑆)
t₀ = 2.0
z(t₀)
-2.4289272381356413 + 3.6879864883532325im

or over a step range of time instants.

using ISA

C₀ = AMFMtriplet(t->cos.(t),ω->100,0.1)
C₁ = AMFMtriplet(t->2*t,ω->10,1.0)

𝑆 = compSet([C₀,C₁])
z = AMFMmodel(𝑆)
t = 0.0:0.25:1.0
z(t)
5-element Vector{ComplexF64}:
   0.9950041652780258 + 0.09983341664682815im
   0.5001647931840112 - 0.2071093295078724im
   1.8257641894671868 - 0.4239781086499489im
 -0.20362579160283556 + 0.9827438231711381im
  0.49975012407155917 - 2.2256904493884333im

Another example of evaluating an AM–FM model over a range of time instants using the Plots module follows.

using Plots
t = 0.0:0.005:2.0
p1 = plot(t, real(z(t)), xlab="t", ylab="real", legend = :false)
p2 = plot(t, imag(z(t)), xlab="t", ylab="imag", legend = :false)
plot(p1, p2, layout = (2,1))
